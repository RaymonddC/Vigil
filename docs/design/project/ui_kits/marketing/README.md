# Vigil Marketing — UI Kit

Public landing page (`/`) for first-time visitors. Pitches Vigil to nurses, residents, CNOs, and CMIOs.

Open `index.html`. Single-file — marketing pages don't need component decomposition at this scale. The styles (`marketing.css`) sit on top of the root `colors_and_type.css` tokens.

**Sections.**
1. Hero — "A second pair of eyes on every post-op bed." + live roster preview
2. Principles — HITL, information density, cross-ward
3. Anatomy — the SBAR card as the product's central artifact
4. Security bar — HIPAA, SOC 2, FHIR R4
5. Footer

The CTAs link into `../app/index.html` so the kit is navigable as a pair.
